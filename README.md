🌐 Site Pessoal – Projeto PW III
📌 Sobre o projeto

Este projeto foi desenvolvido como parte das atividades da disciplina Programação Web III (PW III), orientada pelo professor André Olímpio.

A proposta do trabalho é apresentar um site pessoal criado com HTML, aplicando práticas utilizadas em projetos reais de desenvolvimento, como:

utilização de repositório no GitHub

inclusão de arquivos LICENSE, .gitignore e README

organização adequada da estrutura de pastas

boas práticas na escrita e separação do código

Além da parte técnica, o site também apresenta um pouco da minha trajetória acadêmica, mostrando meus interesses, estudos e desenvolvimento na área de tecnologia.

🎯 Objetivo

Colocar em prática os conceitos fundamentais de HTML

Utilizar CSS e JavaScript para melhorar o design e a interatividade do site

Aplicar práticas de organização e versionamento de código

Desenvolver um projeto com estrutura semelhante à utilizada no ambiente profissional

🛠️ Tecnologias utilizadas

HTML5 – responsável pela estrutura e conteúdo do site

CSS3 – utilizado para estilização e organização visual

JavaScript – responsável por funcionalidades interativas da página
